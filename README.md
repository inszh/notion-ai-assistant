# Notion AI 助手

一款专为 Notion 用户打造的划词翻译工具，支持 8 种语言，完美适配 Notion 插入逻辑。

## ✨ 功能特点
- **多语言支持**：简中、繁中、英、日、韩、泰、印尼、越。
- **Notion 深度适配**：翻译结果一键插入选区下方。
- **国际化提示词**：默认提示词随系统语言自动切换。
- **品牌设计**：保持 Zane 同学统一视觉风格。

## 🚀 安装方法
1. 下载本项目并解压。
2. 打开 Chrome `chrome://extensions/`。
3. 开启“开发者模式”。
4. 点击“加载已解压的扩展程序”，选择本项目文件夹。
